- VPN là gì ?

-- VPN ( Virtual Privtae Network ) là một mạng WAN riêng tư được xây dựng trên Internet

-- Có 4 loại VPN:

-> Acces VPN

-> Site-to-side VPN

-> Intranet VPN

-> Extranet VPN

- Mô hình OSI

-- OSI ( Open System Interconnection ) là một kiểu trúc mạng dựa trên chuẩn ISO.

-- Mô hình OSI có 7 tẩng:

-> 1. Physical

-> 2. Data Link

-> 3. Network

-> 4. Transport

-> 5. Session

-> 6. Presentation

-> 7. Application
